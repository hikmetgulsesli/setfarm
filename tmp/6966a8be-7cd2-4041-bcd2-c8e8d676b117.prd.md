## 7. Renk Sistemi (oklch + Tailwind)

### 7.1 Dark Theme (varsayılan)
- **Background:** oklch(0.15 0 0) — koyu gri-siyah
- **Surface:** oklch(0.20 0 0) — biraz açık
- **Text Primary:** oklch(0.95 0 0 0) — beyaz
- **Text Secondary:** oklch(0.70 0 0 0) — gri
- **Accent Primary:** oklch(0.70 0.15 200) — cyan
- **Accent Secondary:** oklch(0.70 0.15 170) — teal
- **Border:** oklch(0.30 0 0)

### 7.2 Light Theme (toggle)
- **Background:** oklch(0.98 0 0)
- **Surface:** oklch(0.95 0 0)
- **Text Primary:** oklch(0.15 0 0)
- **Text Secondary:** oklch(0.45 0 0)

## 15. Mimari Kararlar

### Karar: Statik Veri vs CMS
- **Tercih:** Statik TypeScript veri dosyaları
- **Gerekçe:** Kişisel site, düşük güncelleme sıklığı, basitlik, hız
- **Alternatif:** CMS (Contentful, Sanity) — gereksiz karmaşıklık

### Karar: Tailwind CSS v4 vs v3
- **Tercih:** Tailwind v4 (CSS-first config)
- **Gerekçe:** Next.js 14 ile uyum, daha basit setup, CSS-native değişkenler
- **Alternatif:** Tailwind v3 — daha olgun ama daha fazla config

### Karar: shadcn/ui
- **Tercih:** shadcn/ui (manual install, copy-paste)
- **Gerekçe:** Tam kontrol, no abstraction layer, customizability
- **Alternatif:** MUI, Chakra — daha ağır, daha az custom
